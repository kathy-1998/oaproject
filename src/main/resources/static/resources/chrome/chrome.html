Axure RP 为 Chrome的扩展插件

Chrome 或者类似基于Chrome开发的浏览器（比如360、猎豹等等）需要安装这个插件查看本地存储的原型。安装之后你会喜欢的, 您可以使用其他浏览器或者把原型发布到WEB服务器上比如 AxShare.

 

1. 安装插件 (免费)

点击这里进行安装

2. 打开Chrome浏览器的扩展程序页面

 

3. 选择 "允许访问文件网址"

4. 查看原型

点击这里打开原型

扩展程序常见问题

什么是Chrome扩展程序？
 扩展额外的特性和功能，可以添加到谷歌浏览器。

为什么我需要安装扩展？
 这个扩展只是在用Chrome查看本地驱动器上存储的原型文件需要。原型文件放置在WEB服务器上着可以不用安装扩展应用。

为什么这个扩展需要一个高级别的访问权限？
 访问是必需的，因为原型的脚本和框架在本地文件夹内，通过file:// 协议（本地文件）访问。您大可放心，AXURE没有跟踪或访问您的任何信息。

我没有互联网连接的情况下怎么安装？
 (步骤 1)在Chrome中打开程序扩展（chrome://extensions/）， (步骤2). 然后打开windows资源管理器或者Finder 拖动原型目录下的"resources/chrome/axure-chrome-extension.crx" 的文件到程序扩展即可进行安装。
